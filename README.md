# ☁️ Electron Video File Converter

Convert your favourite 📼s to:
 - AVI
 - M4V
 - MOV
 - MP4
 - MPEG
 - or OGV

with the super easy to use drag and drop feature from React Dropzone

# Installation

`npm install`

Start dev server:

`npm start`

Start Electron:

`npm run electron`
